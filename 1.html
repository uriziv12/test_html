הוכחת נוסחת השורשים לפתרון משוואה ריבועית

המשוואה הריבועית הכללית

ax² + bx + c = 0

כאשר a, b ו-c הם קבועים ו-a ≠ 0.

מטרה

המטרה היא למצוא את ערכי x המקיימים את המשוואה, כלומר את שורשי המשוואה.

פתרון

1. חילוק ב-a

נחלק את שני אגפי המשוואה ב-a:

x² + (b/a)x + (c/a) = 0

2. העברת c/a לאגף ימין

נעביר את האיבר c/a לאגף ימין:

x² + (b/a)x = -c/a

3. השלמה לריבוע

נוסיף וגם נפחית את (b/2a)² לשני האגפים:

x² + (b/a)x + (b/2a)² = -c/a + (b/2a)²

4. פישוט אגף שמאל

אגף שמאל הוא כעת ריבוע מושלם:

(x + b/2a)² = -c/a + (b/2a)²

5. פישוט אגף ימין

נפשט את אגף ימין על ידי מציאת מכנה משותף ואיחוד השברים:

(x + b/2a)² = (b² - 4ac) / 4a²

6. הוצאת שורש ריבועי

נוציא שורש ריבועי משני האגפים (זכרו שיש שני שורשים אפשריים):

x + b/2a = ±√(b² - 4ac) / 2a

7. בידוד x

נבודד את x על ידי העברת b/2a לאגף ימין:

x = -b/2a ± √(b² - 4ac) / 2a

8. איחוד שברים

נאחד את שני השברים:

x = (-b ± √(b² - 4ac)) / 2a

נוסחת השורשים

x = (-b ± √(b² - 4ac)) / 2a

נוסחה זו נותנת את שני הפתרונות האפשריים (השורשים) של המשוואה הריבועית.
