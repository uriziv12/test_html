הוכחת נוסחת השורשים לפתרון משוואה ריבועית

המשוואה הריבועית הכללית

$$ax^2 + bx + c = 0$$

כאשר a, b ו-c הם קבועים ו-a ≠ 0. מדוע a חייב להיות שונה מאפס? אם a היה שווה לאפס, המשוואה לא הייתה ריבועית אלא קווית.

1. חילוק ב-a

נחלק את שני אגפי המשוואה ב-a:

$$x^2 + \frac{b}{a}x + \frac{c}{a} = 0$$

חשוב: פעולה זו מותרת משום ש-a ≠ 0.

2. העברת c/a לאגף ימין

נעביר את c/a לאגף ימין:

$$x^2 + \frac{b}{a}x = -\frac{c}{a}$$

3. השלמה לריבוע

נוסיף $$(\frac{b}{2a})^2$$ לשני האגפים כדי להשלים לריבוע:

$$x^2 + \frac{b}{a}x + (\frac{b}{2a})^2 = -\frac{c}{a} + (\frac{b}{2a})^2$$

4. פישוט

נפשט את שני האגפים:

$$(x + \frac{b}{2a})^2 = \frac{b^2 - 4ac}{4a^2}$$

5. הוצאת שורש

נוציא שורש משני האגפים:

$$x + \frac{b}{2a} = \pm\sqrt{\frac{b^2 - 4ac}{4a^2}}$$

6. פישוט השורש

נפשט את השורש:

$$x + \frac{b}{2a} = \pm\frac{\sqrt{b^2 - 4ac}}{2a}$$

7. בידוד x

נבודד את x:

$$x = -\frac{b}{2a} \pm \frac{\sqrt{b^2 - 4ac}}{2a}$$

8. איחוד שברים

נאחד את שני השברים:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

נוסחת השורשים

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

נוסחה זו נותנת את שני הפתרונות האפשריים (השורשים) של המשוואה הריבועית.
